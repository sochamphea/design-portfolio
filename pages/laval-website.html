Website UX Design 2024

Context & Problem

The city of Laval aimed to undertake a complete redesign of its website to align with modern standards, integrating eco-design principles and inclusive writing while preserving existing content.

As a UX Designer, I continued the requirements gathering phase that had already been initiated, working closely with the Project Manager and Content Strategist. Together, we followed a structured, goal-oriented work plan to ensure a coherent and user-centered redesign approach.

Tools

Miro board
Figma

As a Designer I did...

  • Presentation of a detailed work plan with regular co-creation workshops with the design team
  • Benchmark analysis (various municipalities and diverse websites)
  • Information architecture and detailed site mapping
  • Wireframes: Homepage and a detailed key user journey selected based on the citizen experience
  • Preparation of a user testing protocol, including conducting citizen user tests
